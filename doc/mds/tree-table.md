
## TreeTable 树形表格
### 安装
 使用 npm:
```bash
$ npm i vue-table-with-tree-grid -S
```
或者 yarn:
```bash
$ yarn add vue-table-with-tree-grid
```
### 引入项目
```javascript
import ZkTable from 'vue-table-with-tree-grid'

Vue.use(ZkTable)
```
### DEMO
:::demo
```html
  <ty-tree-table/>
  <file>tree-table</file>
```
